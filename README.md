## 📊 Project Overview

This project helps telecom companies identify customers likely to leave their services.

The application analyzes:
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Tech Support
- Senior Citizen Status

and predicts churn probability with business insights.
